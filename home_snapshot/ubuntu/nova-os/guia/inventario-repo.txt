Inventario del Repositorio Nova OS
=================================

Alcance
--------
- Total documentado: 532 archivos operativos del repositorio.
- Este inventario excluye dependencias de terceros y artefactos de build y cache como node_modules, dist, build, venv, __pycache__, data y caches temporales.
- El objetivo es dejar una guia textual util para humanos y agentes, no una copia de dependencias ajenas al proyecto.

Mapa de carpetas
----------------
- ./ -> 36 archivos. Raiz del repositorio con documentos base, manifiestos, variables de entorno y orquestacion general.
- .claude/ -> 2 archivos. Memoria de trabajo e handoffs de sesiones anteriores.
- backend/ -> 183 archivos. Arbol legacy o de compatibilidad con backend historico, empaquetado y una copia del runtime modular.
- docker/ -> 3 archivos. Definiciones de contenedores y compose para desarrollo o despliegue.
- docs/ -> 4 archivos. Documentacion de referencia del producto, arquitectura y despliegue.
- frontend/ -> 140 archivos. Frontend activo React y Vite con branding y publicos.
- guia/ -> 3 archivos. Documentacion operativa local escrita en texto plano.
- legacy/ -> 2 archivos. Material o snapshots heredados separados del runtime principal.
- n8n-nodes-nova/ -> 6 archivos. Paquete comunitario Nova OS para n8n.
- nginx/ -> 1 archivos. Configuracion de Nginx para servir el frontend.
- nova/ -> 127 archivos. Runtime modular principal de Nova OS.
- nova_rules/ -> 7 archivos. Reglas y presets textuales de gobernanza.
- scripts/ -> 4 archivos. Scripts auxiliares de migracion, seed y verificacion.
- tests/ -> 14 archivos. Pruebas del runtime y de integraciones clave.
